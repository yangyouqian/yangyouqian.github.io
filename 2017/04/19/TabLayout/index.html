TabLayout

发表于 2017-04-19 | 分类于 Android UI | 阅读次数

是什么

TabLayout provides a horizontal layout to display tabs.

Tablayout提供横向布局显示标签。
通常与ViewPager一起使用。

它的可使用范围很广泛，在很多应用中都有使用，如上图是掘金的首页界面，红框1和2中的内容都可以用Tablayout+Viewpager实现。

怎么使用

首先需要在gradle文件中添加依赖:

compile 'com.android.support:design:24.2.1'

记得把这个版本号换成自己gradle的版本号。

需要知道的一些使用方法

  • newTab():创建并返回TabLayout.Tab
  • setText (int resId)：设置显示在选项卡上的文字
  • setIcon (int resId)：设置显示在选项卡上的图标

  • addTab (TabLayout.Tab tab)： 向此布局添加选项卡。该选项卡将添加到列表的末尾。如果这是要添加的第一个选项卡，它将成为选定的选项卡。

    TabLayout tabLayout = ...;
tabLayout.addTab(tabLayout.newTab().setText("Tab 1"));

  • 上面是创建tab的一种方法，还有一种静态的创建方法，使用TabItem在布局文件中创建，直接添加TabItem，属性值text设置自己想要的显示文字，icon设置为要显示的图标，也可以引用布局，用属性layout,代码如下

    <android.support.design.widget.TabLayout
android:layout_width="match_parent"
android:layout_height="wrap_content">
    <android.support.design.widget.TabItem
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:layout="@layout/tabitem"/>
    <android.support.design.widget.TabItem
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:text="tab2"/>
    <android.support.design.widget.TabItem
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:icon="@mipmap/ic_launcher"/>
</android.support.design.widget.TabLayout>

//tabitem的布局
<RelativeLayout xmlns:android="http://schemas.android.com/apk/res/android"
android:layout_width="match_parent"
android:layout_height="match_parent">
    <TextView
        android:layout_width="wrap_content"
        android:layout_height="wrap_content"
        android:textSize="20sp"
        android:id="@+id/tv"
        android:text="test"
        android:layout_centerInParent="true"/>
    <ImageView
        android:layout_width="30dp"
        android:layout_height="30dp"
        android:src="@mipmap/ic_launcher"
        android:layout_centerInParent="true"
        android:layout_toLeftOf="@id/tv"/>
</RelativeLayout>
